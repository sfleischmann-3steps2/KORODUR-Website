# Schwellensanierung, Göppingen-Jebenhausen

## Prosa (DE)

Um Spannungen entgegenzuwirken wurde die Schwelle in zwei Abschnitte eingeteilt und bedingt durch unterschiedliche Auftragsstärken mit CEMENT ALL und MORTAR MIX saniert.
Somit konnte im Laufe weniger Stunden die Nutzung der Garage sichergestellt werden.

Für den Unternehmer eine einfache und schnelle Sanierung und für den Bauherren eine sichergestellte, schnelle Nutzungsfähigkeit.

RAPID SET – schnell, einfach, gut!

## Prosa (EN)

To compensate for any tension, the threshold was divided into two parts, and due to the different application thicknesses, repaired with CEMENT ALL and MORTAR MIX. Only a few hours later the garage was accessible again.

A quick and easy renovation for the contractor, and a guaranteed fast re-usability for the building owner.

RAPID SET - fast, easy, unique!
