Anwendungsmatrix mobil: zwei Varianten (PR 6)

Desktop (ab 1024px) behält die Poster-Tabelle 1:1. Darunter ersetzt eines der beiden folgenden Layouts den heutigen 980px-Zwangs-Scroll. Beide Varianten nutzen echte Daten aus data/anwendungsmatrix.ts und shadcn-Bausteine (Card, Accordion, Badge, Button).

Empfehlung: Variante B als Standard-Einstieg (Anwendung zuerst, so denkt der Verarbeiter), Variante A als „Alle Produkte"-Ansicht darunter bzw. per Umschalter. Technisch sind beide derselbe Datenstand, nur anders gruppiert.
Variante A

Produkt-Karten + Accordion

Eine Karte je Produkt. Anwendungen als Liste (✓✓ Kernanwendung, ✓ geeignet), technische Daten eingeklappt.
