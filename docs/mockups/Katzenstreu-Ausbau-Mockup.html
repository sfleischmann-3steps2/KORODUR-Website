Diskussionspunkte

  1. Farbwelt: Das Mockup nutzt die Katzenstreu-CD-Farbe #1E3A5F (CD Rules) für Hero/Private-Label statt KORODUR-Navy — die Abgrenzung zum Rest der Site wird damit sichtbar, bleibt aber seriös. Die früher angedachten „Grün-Highlights Richtung Heidelberg-Materials-Grün" wären eine dritte Richtung — Entscheidung nötig: #1E3A5F (CD Rules), Grün, oder neutral wie bisher?
  2. Claim-Platzierung: „HAPPY PETS – HAPPY PEOPLE" prominent im Hero — passt das, oder eher dezenter (z. B. nur im Markenblock)?
  3. Benticat-Inhalte: Für den Markenblock fehlen Sortiment, Qualitäten, Gebinde, USPs und bessere Produktfotos (aktuell nur ein Amazon-Bild). Zulieferung nötig — bis dahin Marken-Teaser mit Anfrage-CTA. Gibt es eine benticat.de/Domain?
  4. Messe-Block: Interzoo 2026 war im Mai (Fotos liegen vor). Rückblick zeigen, kommende Termine pflegen (PLMA?) — oder Block weglassen, bis ein Termin ansteht?
  5. Struktur: Eine ausgebaute Bereichsseite (wie hier) oder eigene Unterseiten je Marke (/katzenstreu/goodcat, /katzenstreu/benticat)? Empfehlung: eine Seite, solange Benticat-Inhalte schlank sind.
  6. Icons: Messe/Icons-Katze.pdf enthält USP-Icons (staubarm, klumpend, geruchsbindend …) — für die Produktkarten nutzbar, wenn als SVG/PNG exportierbar.
  7. Kein KORODUR-Logo im Bereich (CD Rules) — aktuell zeigt die Site Header/Footer global. Vorschlag: Header bleibt (Navigation), aber der Bereich verzichtet auf zusätzliche KORODUR-Markierung im Inhalt.
